# Job-MiniProgram
一个 大学生兼职 微信小程序



- 1.0版本已经过去!!!也是让关注项目的小伙伴认识它的版本。1.0版本项目介绍和截图戳链接[Job MiniProgram V1.0回顾](https://github.com/Gang-bb/Job-MiniProgram/wiki/Job-MiniProgram-V1.0%E5%9B%9E%E9%A1%BE)。代码在Tag  `v1.0.0`版本。感兴趣的小伙伴可以看看~
